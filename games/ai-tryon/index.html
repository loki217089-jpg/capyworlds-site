🧥 AI 虛擬試穿是什麼？

你在網路上看到一件衣服，想知道「穿在我身上會是什麼樣子」——這就是 AI 虛擬試穿要解決的問題。

它背後是 Diffusion Model（擴散模型），跟 Stable Diffusion 同系列技術，但專門針對人體姿勢與服裝形變做了訓練。

IDM-VTON OOTDiffusion CatVTON Fashn.ai API

⚙️ 工作流程

①
輸入人物照片（Model Image）
你自己的全身照，站立姿勢效果最好
②
輸入服裝照片（Garment Image）
白底商品圖或平鋪圖，越乾淨越好
③
指定服裝類別（Category）
tops / bottoms / one-pieces 三擇一
④
API 送出 → 非同步等待
模型通常需要 10~30 秒，用輪詢取得結果
⑤
取回結果圖片 URL
AI 合成的「你穿上那件衣服」的圖片

🔑 API 選擇比較

Fashn.ai（推薦新手）

REST API 格式簡單，有免費額度（每月約 20 次），文件清楚。
網址：fashn.ai | 定價：約 $0.05 /次

Replicate（開源模型）

可以跑 IDM-VTON、OOTDiffusion 等開源模型，按秒計費。
適合想要控制力更強的開發者，學習成本稍高。

自架（Hugging Face Spaces / RunPod）

最省錢但最複雜。需要 GPU 機器（T4/A10G），適合有一定基礎的人。

🔒 API Key 安全注意事項

API Key 絕對不能放在前端 JS 裡公開！正確做法是用 後端代理：

前端 → Cloudflare Worker（隱藏 Key）→ Fashn.ai API

這樣 Key 只存在 Worker 環境變數，使用者看不到。本頁的「實際體驗」tab 是讓你自己填 Key 測試用，正式產品不該這樣做。

免費申請：fashn.ai → Dashboard → API Keys。Key 只在你的瀏覽器使用，不會上傳到任何伺服器。

① 前端直接呼叫（測試用）

// 1. 圖片轉 base64 async function toBase64(file) { return new Promise(res => { const r = new FileReader(); r.onload = e => res(e.target.result); r.readAsDataURL(file); }); } // 2. 送出換裝請求 async function tryOn(personFile, garmentFile, category, apiKey) { const [person, garment] = await Promise.all([ toBase64(personFile), toBase64(garmentFile) ]); // 提交任務 const res = await fetch('https://api.fashn.ai/v1/run', { method: 'POST', headers: { 'Authorization': `Bearer ${apiKey}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ model_image: person, // data:image/jpeg;base64,... garment_image: garment, category: category // "tops" | "bottoms" | "one-pieces" }) }); const { id } = await res.json(); // 輪詢等待結果（最多等 60 秒） for (let i = 0; i < 30; i++) { await new Promise(r => setTimeout(r, 2000)); // 等 2 秒 const poll = await fetch(`https://api.fashn.ai/v1/status/${id}`, { headers: { 'Authorization': `Bearer ${apiKey}` } }); const data = await poll.json(); if (data.status === 'completed') return data.output[0]; // 圖片 URL if (data.status === 'failed') throw new Error(data.error); } throw new Error('Timeout'); } // 使用範例 const imgUrl = await tryOn(personFile, garmentFile, 'tops', 'fa-xxx...');

② Cloudflare Worker 代理（正式產品）

// worker/index.js — 在 wrangler.jsonc 加環境變數 FASHN_KEY export default { async fetch(request, env) { const url = new URL(request.url); // CORS const cors = { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'POST,GET,OPTIONS', 'Access-Control-Allow-Headers': 'Content-Type' }; if (request.method === 'OPTIONS') return new Response(null, { headers: cors }); // 代理 /tryon → fashn.ai/v1/run if (url.pathname === '/tryon' && request.method === 'POST') { const body = await request.json(); const resp = await fetch('https://api.fashn.ai/v1/run', { method: 'POST', headers: { 'Authorization': `Bearer ${env.FASHN_KEY}`, 'Content-Type': 'application/json' }, body: JSON.stringify(body) }); const data = await resp.json(); return Response.json(data, { headers: cors }); } // 代理 /tryon-status/:id → fashn.ai/v1/status/:id const m = url.pathname.match(/^\/tryon-status\/(.+)$/); if (m) { const resp = await fetch(`https://api.fashn.ai/v1/status/${m[1]}`, { headers: { 'Authorization': `Bearer ${env.FASHN_KEY}` } }); return Response.json(await resp.json(), { headers: cors }); } return new Response('Not found', { status: 404 }); } } // wrangler.jsonc 加： // "vars": { "FASHN_KEY": "fa-你的key" }

③ 前端使用 Worker 代理版

// 不需要 API Key！Key 藏在 Worker 裡 const WORKER = 'https://your-worker.your-name.workers.dev'; async function tryOnViaWorker(personFile, garmentFile, category) { const [person, garment] = await Promise.all([ toBase64(personFile), toBase64(garmentFile) ]); const { id } = await fetch(`${WORKER}/tryon`, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ model_image: person, garment_image: garment, category }) }).then(r => r.json()); for (let i = 0; i < 30; i++) { await new Promise(r => setTimeout(r, 2000)); const data = await fetch(`${WORKER}/tryon-status/${id}`).then(r => r.json()); if (data.status === 'completed') return data.output[0]; if (data.status === 'failed') throw new Error(data.error); } }
