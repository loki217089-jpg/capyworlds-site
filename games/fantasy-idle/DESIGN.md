# 遺跡紀行 Relic Chronicle — 設計文件

## 核心概念

**一句話**：「探索六大奇幻秘境，蒐集失落文物，組建探險隊，揭開世界的秘密。」

**類型**：奇幻復古放置 RPG（探索蒐集型）
**靈感來源**：Your Chronicle（深度機制）+ 考古探險 + 博物館蒐集

## 競品差異化

| Your Chronicle | 遺跡紀行 |
|---------------|---------|
| 純文字介面 | 像素風 Canvas + 場景動畫 |
| 轉生 Prestige 核心 | **探索蒐集**核心（無轉生） |
| 5+ 種資源 | 3 資源（金幣/經驗/遺物碎片） |
| 召喚怪物 | 招募探險隊員（5 種職業） |
| 漸進解鎖自動化 | 第 1 分鐘就有自動探索 |

## 世界觀

遠古時代，六大秘境封印著失落文明的遺物。
你是最後的遺跡獵人，帶領探險隊穿越森林、沙漠、雪山、深海、浮空島、龍脊火山，
蒐集所有遺物，解開世界的真相。

## 核心迴圈

```
選擇秘境 → 自動探索（戰鬥 + 發現）→ 獲得金幣/經驗/碎片
  ↓                                      ↓
解鎖新秘境 ← 升級 ←──────────── 強化隊員/裝備
  ↓
蒐集文物 → 填滿圖鑑 → 圖鑑獎勵（全域加成）
```

## 探險隊系統

5 種職業，最多上陣 4 人：

| 職業 | 特長 | 加成 |
|------|------|------|
| 戰士 Warrior | 近戰坦克 | +HP, +防禦 |
| 弓手 Ranger | 遠程輸出 | +攻擊, +暴擊 |
| 法師 Mage | 魔法 AOE | +魔攻, +經驗 |
| 學者 Scholar | 鑑定文物 | +碎片掉落, +文物發現率 |
| 工匠 Engineer | 製造裝備 | +金幣, +裝備品質 |

## 文物圖鑑系統

- 每個秘境 8 件文物，分 N/R/SR/SSR 四個稀有度
- 探索時隨機發現（稀有度越高機率越低）
- 蒐集同秘境全套 → 解鎖套裝加成（全域永久）
- 圖鑑完成度影響標題畫面裝飾

| 文物 | 稀有度 | 說明 |
|------|--------|------|
| 苔蘚之劍 | N | 被苔蘚覆蓋的古代長劍 |
| 精靈弓 | N | 刻有精靈文字的短弓 |

| 月光寶石 | SR | 只在滿月夜閃耀的寶石 |
| 世界樹枝 | SR | 傳說中世界樹的一根枝條 |
| 精靈王權杖 | SSR | 統治整片森林的至高權杖 |

## 戰鬥系統

- **全自動**：隊員自動攻擊，玩家不需操作
- 每個秘境有 3 種怪物 + 1 個 Boss
- Boss 每 10 次探索出現一次
- 擊敗 Boss 有額外碎片 + 文物掉率加成
- 戰鬥以文字 log + 血條動畫呈現（不需複雜動畫）

## 商店系統

| 項目 | 價格 | 效果 |
|------|------|------|
| 鐵劍 → 鋼劍 → 魔法劍 | 金幣 | +攻擊力 |
| 皮甲 → 鎖甲 → 板甲 | 金幣 | +防禦力 |
| 探索地圖 | 碎片 | +文物發現率 10% |
| 幸運護符 | 碎片 | +稀有度提升 5% |
| 經驗卷軸 | 金幣 | +經驗獲取 15% |

## 美術風格

- **風格**：CSS + Canvas 像素感（無外部圖片）
- **配色**：深色背景 + 金色強調，復古 RPG 感
- **場景**：Canvas 繪製（樹/沙丘/雪山/海底/雲/火山）
- **怪物**：emoji + Canvas 簡筆（眼睛+身體形狀）
- **文物**：emoji 圖示 + 發光特效
- **字體**：sans-serif（中英相容）
