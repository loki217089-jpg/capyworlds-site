## 差異化策略

| 競品缺點 | 我們的解法 |
|----------|-----------|
| 不存檔 | **localStorage 自動存檔**（升級/解鎖/最高波次） |
| 廣告多 | **零廣告**（自有網站，CrazyGames revenue share） |
| 後期空虛 | **Prestige 系統** — 重置換 Meta 點數，解鎖新角色/箭種 |
| 箭種弱 | **12+ 箭種**，每種有明顯不同的物理行為和視覺效果 |
| 操控飄 | **精準拉弓**：拖曳距離 = 力道，角度 = 方向，加上輔助瞄準線 |
| 玩久重複 | **波次系統 + 環境變化** — 每 5 波換場景，有風/雨/晃動等干擾 |

**獨特賣點（一句話）**：「布娃娃弓箭手 + Roguelike 升級 + 物理爽感 = 停不下來的射擊生存」

## 核心玩法設計

### 一句話概念
> 玩家拉弓射箭，擊敗一波波布娃娃敵人。每次命中都有物理回饋，賺金幣升級，活越久越強。

### 遊戲循環

```
拉弓射擊 → 命中敵人（物理飛散 + 傷害數字）
     ↓
敵人射你 → 閃避 or 被命中（身體各部位判定）
     ↓
殺敵 → 掉落金幣/道具 → 波次間升級
     ↓
Boss 波 → 高獎勵 → 解鎖新箭種
     ↓
死亡 → Prestige? → 用 Meta 點數永久強化 → 再來一局
```

### 操控方式

| 平台 | 操作 |
|------|------|
| 電腦 | 滑鼠拖曳拉弓（類似彈弓），Space 跳躍 |
| 手機 | 觸控拖曳拉弓，雙指跳躍 or 按鈕跳躍 |

- 拖曳距離 → 射擊力道（有力道條顯示）
- 拖曳方向 → 射擊角度（反向，像拉彈弓）
- 長按太久 → 消耗體力（防止無限蓄力）
- 放開 → 發射

## 技術規劃

| 項目 | 方案 |
|------|------|
| 渲染 | Canvas 2D |
| 物理 | 自寫簡易 ragdoll（關節 + 約束 + verlet 積分） |
| 碰撞 | 箭矢用線段 vs 身體部位用圓形 |
| 動畫 | 布娃娃 = 物理驅動，不需 sprite 動畫 |
| 音效 | 素材庫（拉弓/放箭/命中/爆頭/死亡） |
| 存檔 | localStorage |
| 語言 | 繁中 + EN 雙語 |
| 預估行數 | ~1200-1500 行（需分塊寫入） |

### 物理引擎核心

```
Ragdoll = 8 個質點（頭、軀幹上下、左右手上下、左右腳上下）
         + 骨骼約束（固定距離）
         + 重力 + 碰撞

每幀更新：
1. 施加重力
2. Verlet 積分更新位置
3. 約束求解（3-5 次迭代）
4. 地面碰撞
5. 箭矢碰撞判定（線段 vs 圓）
```
