# 🌿 水豚生態塔防 — Capy Ecosystem Defense

> 「種植物、養動物、讓生態系統自己成為防線」

## 核心概念

不放「塔」，放「生物」。植物和動物之間有生態鏈互動（Synergy），
敵人是「污染物」——工廠廢氣、塑膠怪、毒液史萊姆。
水豚是核心支援單位，放在水邊時 buff 全場。

## 地圖設計

- **Canvas 640×640**，格子 20×20（每格 32px）
- 固定 S 型路線，敵人從左上進、右下出
- 地形：🟩 草地（可放置）、🟦 水域（水豚 buff 觸發）、🟫 路徑（敵人走）
- 路線旁有 2 格水域，玩家要策略性放水豚

## Roguelike 層：波次間升級

每 3 波結束後，從 3 張隨機卡中選 1 張：

- 「光合加速」— 所有向日葵 +30% 產出
- 「尖刺突變」— 荊棘叢傷害 ×2
- 「蜂群暴怒」— 蜂巢攻速 +50%
- 「狼王降臨」— 所有狼 +25% 傷害
- 「生態平衡」— 全場所有單位回復 30% HP
- 「營養土壤」— 養分掉落 +50%
- 「水豚之力」— 水豚 buff 範圍擴大至全場 +30%
- 「森林之盾」— 所有植物 +40% HP
- 「獵食本能」— 所有動物 +20% 攻速

## 技術規格

- 單一 `index.html`，CSS+JS inline
- Canvas 640×640 + 右側 UI 面板（選單位、資源顯示）
- 手機：Flexbox 三段式佈局（top-bar / game-area / action-bar）
- 繁中/英文雙語 i18n
- localStorage 存：Meta 升級、最高波次、生態點數
- 素材：32rogues spritesheet + critters 動畫 + Canvas 自繪混合
