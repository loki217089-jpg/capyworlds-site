## 一句話描述
經營荒廢農場，透過小遊戲提升效率，完成村民委託，讓農場從荒蕪走向繁榮。

## 核心循環
```
種田/挖掘/釣魚/採集 → 獲得資源 → 完成委託 → 農場升級(視覺變化) → 解鎖新作物/動物/區域
       ↑                                                              ↓
   小遊戲(高效率) 或 掛機(低效率)                              新的委託任務
```

## 四大小遊戲

### 1. 種田 — 節奏播種
- 土壤上方有移動的種子指示器，按對時機播種
- Perfect/Good/Miss 三級判定
- Perfect = 生長速度 ×2、Good = ×1.5、掛機 = ×1

### 2. 挖掘 — 打地鼠式
- 地面冒出石頭/礦石，限時內點擊收集
- 連續命中有 combo 加成
- 掛機 = 每分鐘固定產出

### 3. 釣魚 — 經典釣魚條
- 移動的魚影，按住保持在綠色區間
- 魚會掙扎移動，越稀有越難抓
- 掛機 = 只能釣到普通魚

### 4. 採集 — 滑動收割
- 畫面出現成熟作物/果實，滑動收割
- 速度越快分數越高
- 掛機 = 慢慢自動收

## 生態系統

### 互動關係
```
雞 ──吃蟲──→ 保護作物（減少蟲害）
牛 ──產糞──→ 肥料（加速生長 +20%）
羊 ──吃草──→ 清理雜草（騰出種植空間）
鴨 ──吃蝸牛──→ 保護菜園
豬 ──翻土──→ 挖出隱藏種子/礦石
```

### 天氣系統
| 天氣 | 效果 |
|------|------|
| 晴天 | 正常生長 |
| 雨天 | 生長 +30%，但蟲害 +50% |
| 乾旱 | 生長 -50%，需要澆水 |
| 暴風 | 作物可能損毀，動物躲進棚子 |

### 平衡機制
- 養太多動物 → 飼料不夠 → 動物生病
- 種太多作物 → 蟲害爆發 → 需要更多雞
- 不種草 → 羊沒東西吃 → 羊逃跑

## 農場視覺階段

| 階段 | 條件 | 視覺變化 |
|------|------|---------|
| 0 荒蕪 | 初始 | 雜草叢生、破舊柵欄、灰暗色調 |
| 1 起步 | 完成3個任務 | 清除部分雜草、修好柵欄 |
| 2 成長 | 完成8個任務 | 有作物田、1-2隻動物、風車開始轉 |
| 3 繁榮 | 完成15個任務 | 多種作物、動物滿圈、煙囪冒煙、花朵裝飾 |
| 4 夢幻 | 完成全部任務 | 金色光暈、彩虹、所有裝飾滿載 |

## 委託任務（範例）

| 任務 | 需求 | 獎勵 |
|------|------|------|
| 幫牛找回小牛 | 釣魚小遊戲得分100+ | 解鎖牛 |
| 修好風車 | 收集 5 石頭 + 3 木材 | 風車開始轉動（視覺） |
| 種出巨型南瓜 | 南瓜 Perfect 種植 3 次 | 農場裝飾 |
| 羊毛大豐收 | 養 3 隻羊，草地 > 50% | 解鎖紡織 |
| 消滅蟲害 | 養 5 隻雞 | 永久減少蟲害 |
| 修復穀倉 | 收集 10 木材 + 5 石頭 | 倉庫容量 ×2 |
