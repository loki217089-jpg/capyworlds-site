## 遊戲引用路徑

所有遊戲位於 `games/<name>/index.html`，引用素材的相對路徑為：

```
../../assets/<素材路徑>
```

例如：
```javascript
// BGM
startBGM('../../assets/music/JDSherbert - Minigame Music Pack [FREE]/JDSherbert - Minigame Music Pack - Beach Vibes.ogg');

// SFX
loadAudio('coin', '../../assets/sfx/400 Sounds Pack/Items/coin_collect.wav');

// 圖片
const img = new Image();
img.src = '../../assets/images/Icons_Essential/Icons/Coin.png';
```
