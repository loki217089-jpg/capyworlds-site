# Android 競品分析報告（2026/4/1）

> CapyWorlds 對標的 Android 遊戲與應用程式競品分析，含實際數據與差異化策略。

---

## Part 1：遊戲競品（5 款）

### 1. Idle Miner Tycoon — Kolibri Games（放置/Idle）

| 項目 | 內容 |
|------|------|
| 下載量/評分 | 1 億+ / 4.7★ |
| 核心迴圈 | 挖礦→升級礦井/電梯/倉庫→僱經理自動化→解鎖新礦區→Prestige 重置 |
| 商業模式 | 免費+可選廣告（看廣告雙倍離線收益）+Super Cash IAP+禮包。月營收約 $60 萬 |

**優點：**
- 多層策略深度（礦井/電梯/倉庫三角優化）
- 離線進度感強，正向循環養成每天開 App 習慣
- 經理系統提供明確「下一步」目標感

**缺點：**
- 2025 年起廣告越來越侵入，彈出式廣告增多
- 後期不花錢幾乎停滯，被批評 pay-to-progress
- 新礦區玩法大同小異，缺乏驚喜

**CapyWorlds 差異化：**
- 無廣告、純網頁免安裝
- 跨遊戲連動（地球再生↔虛空領航員）創造生態系黏著度
- 水豚 IP 比通用礦工更有辨識度

### 2. Royal Match — Dream Games（三消/Puzzle）

| 項目 | 內容 |
|------|------|
| 下載量/評分 | 3 億+ / 4.7★ |
| 核心迴圈 | 三消過關→獲星星→裝飾皇宮→解鎖新區域→難度提升 |
| 商業模式 | **零廣告**純 IAP。月營收約 **$1.3 億**，累計超 $30 億 |

**優點：**
- 零廣告體驗，沉浸感極高，反而提高付費意願
- Super Light Ball 單一設計讓收入翻 1.5 倍
- 持續活動更新（Royal Pass、Hidden Temple）

**缺點：**
- 「拯救國王」廣告與實際遊戲嚴重不符
- 後期難度刻意提升，soft pay-wall 明顯
- 帳單爭議：重複扣款/超額收費

**CapyWorlds 差異化：**
- 農場三消加入 Gacha（N/R/SR/SSR），收集動力更強
- 卡片技能帶來策略深度
- 無體力限制，想玩就玩

---

### 3. Soul Knight — ChillyRoom（Roguelike/動作）

| 項目 | 內容 |
|------|------|
| 下載量/評分 | ~9,400 萬 / 4.6★（162 萬則評論） |
| 核心迴圈 | 選角色→隨機地牢→雙搖桿射擊→撿武器→打 Boss→解鎖新角色 |
| 商業模式 | 免費+可選廣告+角色/皮膚 IAP（$1-3）。不 pay-to-win |

**優點：**
- 100+ 武器讓每次 run 不同，重玩價值極高
- 本地和線上多人合作
- 持續更新 9 年（2017 至今），2026 新增 PVP
- 公平營利，免費玩家享受完整體驗

**缺點：**
- Origin 模式命中偏移 Bug 持續數月
- Season 模式太肝，完成季票後純刷
- 安裝包 634 MB

**CapyWorlds 差異化：**
- 純網頁即開即玩，不需 634 MB 空間
- 跨遊戲 Meta 升級系統
- 斬魔忍者 + 虛空領航員提供多種 Roguelike 子類型

### 4. AFK Journey — Lilith Games（放置 RPG）

| 項目 | 內容 |
|------|------|
| 下載量/評分 | 1,000 萬+ / 4.2★ |
| 核心迴圈 | Gacha 抽卡→組隊→自動戰鬥→探索開放世界→公會/PVP |
| 商業模式 | Gacha+多種高級貨幣+限時禮包+戰鬥通行證。累計超 $1.1 億 |

**優點：**
- 開放世界+放置 RPG，視覺和體驗同類頂尖
- 角色故事深度+IP 聯動（Fairy Tail）
- 前中期不花錢也能愉快遊玩

**缺點：**
- 後期 Pay-to-Win，競技模式差距巨大
- 7-8 種貨幣讓新手困惑
- 被批評為「穿著舒適外衣的掠奪性營利」

**CapyWorlds 差異化：**
- 勇者傳說是純單機 RPG，沒有 Gacha 挫敗感
- 寶石系統更透明，不搞多種貨幣混淆
- 紅線規則：免費玩家也能玩完核心內容

### 5. Vampire Survivors — poncle（Roguelike/Survival）

| 項目 | 內容 |
|------|------|
| 下載量/評分 | 500 萬+ / 4.6★ |
| 核心迴圈 | 選角色→自動攻擊只控制移動→升級選技能→技能組合combo→撐過30分鐘 |
| 商業模式 | **完全免費**+可選廣告+便宜 DLC。開發者堅持「非掠奪性營利」 |

**缺點：**

- 被大量山寨（Survivor.io 下載量遠超原版）

**CapyWorlds 差異化：**
- 可做「水豚 Survivors」用可愛風格差異化
- 網頁即開即玩（VS 本身也從網頁起家）
- 加入更強 RPG 養成層

## Part 2：應用程式競品（5 款）

### 1. Whoscall — Gogolook（防詐/來電辨識）

| 項目 | 內容 |
|------|------|
| 下載量/評分 | 6,900 萬（全平台超 8,000 萬）/ 3.77★ |
| 核心價值 | 來電辨識、詐騙攔截、URL/截圖查證 |
| 商業模式 | Freemium：免費含廣告，Premium 訂閱提供無廣告+進階攔截 |

**優點：**
- 資料庫超 10 億筆號碼，辨識準確率極高
- Google Play 2025「Best Everyday Essential」大獎
- 與刑事局 165、銀行深度合作
- 2025 新增「一鍵查」截圖查證

**缺點：**
- 免費版廣告干擾多，核心功能需升級 Premium
- 耗電、背景佔記憶體

**CapyWorlds 差異化：**
- 零安裝、零權限、零耗電的輕量查證工具
- 教育導向：教用戶「辨識」而非只「攔截」
- 結合租屋糾紛自救包做生活防詐一站式整合

### 應用程式差異化矩陣

| 維度 | 競品共同弱點 | CapyWorlds 機會 |
|------|-------------|----------------|
| 收費模式 | 多數走向付費訂閱，免費功能不斷縮減 | 全功能免費，靠遊戲內購+廣告變現 |
